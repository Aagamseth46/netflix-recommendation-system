# 🎬 Netflix Recommendation System

## 📌 Problem Statement
Built a movie recommendation system using collaborative filtering to suggest movies based on user ratings.

## 🚀 Approach
- Data preprocessing and cleaning  
- Handling sparse matrix  
- SVD-based collaborative filtering  
- Model evaluation using RMSE  

## 🧠 Tech Stack
- Python  
- Pandas, NumPy  
- Surprise (SVD)  
- Matplotlib, Seaborn  

## 📊 Result
The model successfully predicts user ratings and generates personalized movie recommendations.
